## 📖 Первые шаги в приложении

### 1️⃣ На главном экране:
- Вы увидите список портфелей (изначально 3 примера)
- Нажмите на портфель, чтобы открыть его
- Или создайте новый портфель кнопкой "Новый портфель"

### 2️⃣ Создание собственного портфеля:
```
Главный экран → "Новый портфель" →
  Выбрать тип (консервативный/сбалансированный/агрессивный) →
  Портфель создан ✓
```

### 3️⃣ Добавление позиции в портфель:
```
Открыть портфель → "Добавить позицию" →
  Выбрать класс инструмента (Облигации/Акции/Фьючерсы) →
  Выбрать конкретный инструмент →
  Просмотреть расчёты на вкладке "Характеристики" →
  Перейти на вкладку "Добавить позицию" →
  Указать количество и цену →
  "Добавить" → Позиция добавлена ✓
```

## 📊 Типы анализов

### Облигации (ОФЗ и корпоративные)
При выборе облигации вы видите:
- **YTM** - доходность к погашению
- **Current Yield** - текущая доходность
- **Duration** - влияние ставок на цену
- **Convexity** - нелинейность цены
- **Справедливая стоимость** при 3 сценариях процентных ставок

### Акции
При выборе акции вы видите:
- **P/E Ratio** - оценка акции (недооценена/справедливо/переоценена)
- **ROE** - рентабельность компании
- **Дивидендная доходность** - процент дохода в год
- **Payout Ratio** - какую долю прибыли платят дивидендами
- **Уровень риска** - оценка рисков
- **Потенциал роста** - ожидаемый рост при целевых мультипликаторах

### Фьючерсы (3 роли)
При выборе фьючерса вы видите анализ под одну из 3 ролей:

**🛡️ Хеджер** - страховка от рисков:
- Справедливая цена и справедливость оценки
- Хедж рацион (сколько контрактов для защиты)
- Сценарии P&L при разных движениях цены

**📈 Спекулянт** - активная торговля:
- Количество контрактов с рычагом
- Требуемая маржа (капитал на операцию)
- Максимальный убыток (худший сценарий)
- 3 сценария P&L (бычий/базовый/медвежий)
- Risk/Reward ratio

**🔄 Арбитражер** - поиск безрисковой прибыли:
- Basis (разница спот-фьючерс цены)
- Справедливый basis
- Возможность арбитража
- Чистая прибыль после комиссий
- Рекомендуемая стратегия

## 📈 Анализ портфеля

В каждом портфеле 3 вкладки:

### Вкладка "Позиции"
- Список всех позиций в портфеле
- Для каждой видно: количество, цена входа, текущая цена, P&L
- Можно удалить позицию

### Вкладка "Анализ"
- **Общие показатели**: стоимость, базис, прибыль, доходность
- **Распределение активов**: какой % облигаций, акций, фьючерсов
- **Целевое распределение**: к чему нужно стремиться
- **Стратегические характеристики**: экспортёры, дивидендная стратегия, валютный хедж

### Вкладка "Характеристики"
- Подробная информация о портфеле
- Типы портфеля
- Даты создания/изменения
- Целевые пропорции

## 💡 Примеры использования

### Сценарий 1: Консервативный инвестор
```
1. Выбрать портфель "Консервативный портфель"
2. Открыть его и посмотреть на "Анализ"
3. Видно: 55% облигации, 30% акции, 10% кэш, 5% спекуляции
4. Нажать на позицию → посмотреть расчёты доходности
5. Ожидание: 10-14% годовых с низким риском
```

### Сценарий 2: Активный трейдер
```
1. Создать новый портфель "Агрессивный"
2. Добавить позиции в фьючерсы (Si, GZ, BRF)
3. На каждый фьючерс посмотреть сценарий "Спекулянт"
4. Видно: P&L при разных ценовых движениях, требуемая маржа
5. Понимание: какой рычаг использовать, какой риск
```

### Сценарий 3: Дивидендная стратегия
```
1. Создать портфель "Дивидендный доход"
2. Добавить акции с дивидендом: ЛУКОЙЛ, ВТБ, ММК, Газпром
3. На каждую акцию смотреть "Дивидендные показатели"
4. Проверить: дивидендная доходность, payout ratio, устойчивость
5. Видно: какие компании надёжнее, какой ожидаемый доход
```

## 📝 Советы по использованию

### ✅ Делайте так:
- Начните с консервативного портфеля как базис
- Посмотрите расчёты для каждого инструмента перед добавлением
- Используйте дивидендные портфели как источник стабильного дохода
- Ребалансируйте портфель каждые 3-6 месяцев
- Добавляйте позиции на просадках (покупайте дешевле)

### ❌ Не делайте так:
- Не концентрируйте больше 10-15% в одной позиции
- Не используйте слишком большой рычаг в спекуляциях (>3x)
- Не покупайте без анализа - всегда смотрите расчёты
- Не игнорируйте уровень риска акции
- Не забывайте про диверсификацию

## ❓ Частые вопросы

**Q: Откуда берутся данные инструментов?**
A: В текущей версии используются статические примеры. Для реальных данных нужна интеграция с MOEX API.

**Q: Можно ли изменить цены инструментов?**
A: Текущая версия использует фиксированные цены как примеры. В будущем планируется синхронизация с реальным рынком.

**Q: Что значит "YTM"?**
A: Yield to Maturity - доходность облигации до её погашения, с учётом купонов и текущей цены.

**Q: Как правильно использовать фьючерсы?**
A: Выбирайте роль (хеджер/спекулянт/арбитражер) в зависимости от вашей стратегии, смотрите расчёты и сценарии.

**Q: Какой портфель выбрать новичку?**
A: Начните с "Консервативного портфеля" - он более стабильный и позволит понять логику работы приложения.
