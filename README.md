# Портфельный конструктор Московской биржи

## 🎯 Описание проекта

Полнофункциональное Windows приложение на Flutter для профессионального управления портфелями ценных бумаг, доступных на Московской бирже (MOEX). Приложение включает:

✅ **Управление портфелями** - создание и редактирование портфелей трёх типов (консервативный, сбалансированный, агрессивный)

✅ **Анализ облигаций** - расчёт YTM (Yield to Maturity), текущей доходности, дюрации, справедливой стоимости

✅ **Анализ акций** - оценка по P/E, ROE, дивидендной доходности, модель дисконтирования дивидендов (DDM)

✅ **Анализ фьючерсов** - модель Блека-Шоулза для трёх сценариев:
  - **Хеджер** - расчёт хеджирования и защиты от рисков
  - **Спекулянт** - анализ рычагов и сценариев P&L
  - **Арбитражер** - поиск разницы между спот и фьючерс ценами

✅ **Примеры портфелей** - встроенные стратегические портфели с реальными инструментами Мосбиржи

## 📊 Основные возможности

### 1️⃣ Облигации - Профессиональный расчёт доходности

**Поддерживаемые типы:**
- ОФЗ (государственные облигации)
- Корпоративные облигации

**Расчёты:**
```
YTM (Yield to Maturity)          → Доходность к погашению
Current Yield                     → Текущая доходность
Modified Duration                 → Влияние ставок на цену
Convexity                        → Нелинейность цены
Fair Price Scenarios             → Справедливая цена при разных ставках
  - Бычий (ставка 5%)
  - Базовый (ставка 8%)
  - Медвежий (ставка 10%)
```

**Пример расчёта YTM:**
Используется метод Ньютона-Рафсона для точного расчёта доходности к погашению на основе текущей цены облигации.

### 2️⃣ Акции - Фундаментальный анализ

**Метрики оценки:**
```
P/E Ratio                        → Оценка (недооценена/справедлива/переоценена)
ROE (Return on Equity)           → Эффективность компании
Dividend Yield                   → Доходность по дивидендам
Payout Ratio                     → Доля дивидендов от прибыли
Risk Level                       → Уровень риска (низкий/средний/высокий)
```

**Специальные функции:**
```
DDM Model (Gordon Growth)        → Справедливая стоимость по дивидендам
Growth Potential                 → Потенциал роста при целевых мультипликаторах
Dividend Sustainability          → Надёжность выплат дивидендов
Exporter Mark                    → Маркер для стратегии валютного хеджа
```

**Пример анализа:**
- ЛУКОЙЛ: P/E=12.1 (справедливо), дивиденд=10.5% (высокая доходность), ROE=23.5% (отличная эффективность)

### 3️⃣ Фьючерсы - Модель Блека-Шоулза (3 сценария)

#### 🛡️ Хеджер (Hedger)
Используется для страховки от неблагоприятных движений цены.
```
Справедливая цена       F = S × e^(r×T)
Хедж рацион             Количество контрактов для полной страховки
Стоимость хеджа         Цена страховки
Защита на 1%            Количество в рублях при 1% движении
Сценарии P&L            Прибыль/убыток при -10%, -5%, 0%, +5%, +10%
```

#### 📈 Спекулянт (Speculator)
Активная торговля с рычагом для максимизации доходности.
```
Контрактов с рычагом    Количество позиций с учётом leverage
Требуемая маржа         Капитал, необходимый для открытия позиции
Максимальный убыток     Максимальный риск в деньгах
Сценарии P&L:
  - Бычий               Рост на 1 волатильность
  - Базовый             Без движения цены
  - Медвежий            Падение на 1 волатильность
Risk/Reward             Соотношение рисков и награды
```

#### 🔄 Арбитражер (Arbitrageur)
Поиск безрисковой прибыли от разницы цен.
```
Basis                   Разница спот-фьючерс цены
Fair Basis              Справедливая разница
Arbitrage Mispricing    Возможность для заработка
Стоимость операции      Комиссии и спреды
Чистый арбитраж         Прибыль после комиссий
Рекомендуемая стратегия:
  - Cash-and-Carry      Купить спот, продать фьючерс
  - Reverse C&C         Короткая спот, покупка фьючерса
```

- Наличие валютного хеджа

## 💡 Примеры стратегий

### Стратегия 1: Дивидендный портфель
```
Состав:
- 60% облигации (ОФЗ + корпораты) → купон 7-9% годовых
- 30% дивидендные акции (ЛУКОЙЛ, Газпром, ВТБ) → дивиденд 8-12% годовых
- 10% кэш

Ожидание: 10-14% годовых в рублях
Риск: умеренный (-15-25% на просадках)
```

### Стратегия 2: Экспортёры + валютный хедж
```
Состав:
- 40% облигации (базис)
- 45% акции-экспортёры (энергетика, металлы)
- 10% фьючерсы Si для страховки рубля
- 5% кэш

Ожидание: При ослаблении рубля на 10% → +15-20% доход
Риск: Геополитика, санкции, дивидендные ограничения
```

### Стратегия 3: Спекулятивный блок
```
Состав:
- 60% облигации (стабильная база)
- 20% базовые акции
- 15% спекулятивные позиции/фьючерсы
- 5% кэш на докупку на просадках

Правила:
- Жёсткие стопы (-2-3% от входа)
- Не усреднять убыточные позиции
- Фиксировать прибыль на +5-10%
- Ребалансировать каждый месяц

Потенциал: 20-40%+ годовых
Риск: Высокий (-30-50% на плохих сценариях)
```

## 🎓 Рекомендации

### Для консервативного инвестора
- Начните с облигаций (ОФЗ)
- Добавьте 30% дивидендных акций
- Используйте примеры расчётов для принятия решений
- Мониторьте YTM - при снижении ставок цены облигаций растут

### Для активного трейдера
- Используйте фьючерсы для спекуляций
- Анализируйте сценарии Спекулянта перед входом
- Ставьте жёсткие стопы (2-3% потерь - выход)
- Не превышайте рычаг 3x

### Для дивидендной стратегии
- Выбирайте акции с дивидендом 8%+ (ЛУКОЙЛ, ВТБ, ММК)
- Проверяйте Payout Ratio (оптимально 30-60%)
- Анализируйте ROE (выше 15% = хороший знак)
- Реинвестируйте дивиденды для компаундирования

### Для валютного хеджа
- Держите 30-40% портфеля в экспортёрах
- Это естественная защита от девальвации рубля
- При необходимости используйте Si для дополнительного хеджа
- Мониторьте basis (разницу между спот и фьючерсом)

## 📊 Математические основы

### YTM (Yield to Maturity) облигаций
```
Метод: Newton-Raphson итеративный поиск
Решение уравнения: ΣCFt/(1+ytm)^t = Цена облигации
где CFt = купоны + номинал
Точность: обычно <0.0001 за 5-10 итераций
```

### DDM Model для акций
```
Fair Value = D₁ / (r - g)
где:
D₁ = текущий дивиденд × (1 + g)
r = требуемая доходность (%)
g = темп роста дивидендов (%)
```

### Блек-Шоулз для фьючерсов
```
Fair Price = S × e^(r×T)
где:
S = текущая спот-цена
r = безрисковая ставка (% годовых)
T = время до экспирации (лет)
```
