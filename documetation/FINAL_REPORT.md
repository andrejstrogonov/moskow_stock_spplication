## 📊 Примеры расчётов

### Облигация ОФЗ-26
```
Параметры:
- Номинал: 1000 ₽
- Купон: 7.5% (полугодовой)
- Цена: 1005 ₽
- До погашения: ~6.3 года

Расчёты:
- Current Yield: 7.46% = (1000 × 0.075) / 1005
- YTM: ~7.3% (метод Ньютона-Рафсона)
- Duration: ~4.5 года (влияние на цену)
- Fair Price при ставке 5%: ~1050 ₽ (доход 4.5%)
- Fair Price при ставке 10%: ~960 ₽ (убыток -4.5%)
```

### Акция ЛУКОЙЛ
```
Параметры:
- Цена: 5850 ₽
- EPS: 485.2 ₽
- Дивиденд: 10.5% годовых
- ROE: 23.5%

Расчёты:
- P/E: 12.1 (справедливо для рынка)
- Дивидендный доход: 10.5% годовых (высокий)
- ROE: 23.5% (отличная эффективность)
- Risk Level: Низкий (P/E умеренный, ROE высокий)
- Потенциал роста: ~15.2% за 3 года
```

### Фьючерс Si (USD/RUB)
```
Параметры:
- Спот: 95.20 ₽
- Цена фьючерса: 95.50 ₽
- Волатильность: 18.5%
- До экспирации: 90 дней
- Risk-free rate: 16%

Расчёты:
- Fair Price (Блек-Шоулз): 95.45 ₽
- Справедливость: +0.05% (правильная цена)
- Для хеджера: Можно хеджировать с минимальной страховкой
- Для спекулянта: Потенциал движения ~1.4% при volatility
- Для арбитражера: Basis = +0.30 ₽ (Cash-and-Carry возможен)
```

## 🎓 Стратегии портфелей

### 1. Консервативный (10-14% годовых)
```
Структура:
- 55% облигации (ОФЗ 7-8%, корпораты 8-9%)
- 30% дивидендные акции (9-11%)
- 10% кэш
- 5% спекуляции

Риск: -15-25% на просадках
Рекомендуется: консервативным инвесторам, пенсионерам
```

### 2. Сбалансированный (15-25% годовых)
```
Структура:
- 35% облигации (базис, доход 7-9%)
- 50% акции (дивиденды + рост)
- 10% кэш на возможности
- 5% фьючерсы (хедж рубля)

Риск: -25-35% на стрессовых событиях
Рекомендуется: активным инвесторам среднего риска
```

### 3. Агрессивный (25-40%+ годовых)
```
Структура:
- 15% облигации (подушка)
- 65% акции (якоря + рост)
- 5% кэш
- 15% фьючерсы (спекуляции)

Риск: -40-50% в плохые периоды
Рекомендуется: опытным трейдерам, высокий риск-tolerance
```

## 🚀 Как использовать

### Быстрый старт (30 секунд)
```powershell
# 1. Перейти в папку проекта
cd C:\Users\strog\StudioProjects\moskow_stock_spplication

# 2. Запустить приложение
.\build\windows\x64\runner\Release\moskow_stock_spplication.exe

# 3. Наслаждаться!
```

### Демонстрация (5 минут)
1. Посмотрите на 3 встроенных портфеля
2. Откройте "Консервативный портфель"
3. Нажмите на позицию "ОФЗ-26" → смотрите YTM, Duration
4. Нажмите на позицию "ЛУКОЙЛ" → смотрите P/E, дивидендный анализ
5. Нажмите на вкладку "Анализ" → смотрите распределение активов

### Создание своего портфеля (10 минут)
1. Нажмите "Новый портфель"
2. Выберите тип или кастомное распределение
3. Нажмите "Добавить позицию"
4. Выберите инструменты (облигации, акции, фьючерсы)
5. Посмотрите расчёты перед добавлением
6. Портфель готов к анализу!
