# Портфельный конструктор Московской биржи

Комплексное Windows приложение для создания и управления портфелями ценных бумаг, доступных на Московской бирже, с профессиональными инструментами анализа и расчётов.

## 🎯 Возможности

### 1. Управление портфелями
- **Создание портфелей** трёх типов: консервативный, сбалансированный, агрессивный
- **Добавление позиций** из трёх классов инструментов: облигации, акции, фьючерсы
- **Анализ портфеля** в реальном времени:
  - Общая стоимость и структура
  - Распределение по типам инструментов
  - Прибыль/убыток по портфелю
  - Характеристики экспортёров и валютный хедж

### 2. Облигации - Расчёт доходности

Полная поддержка расчётов параметров облигаций как ОФЗ, так и корпоративных:

**Основные показатели:**
- **YTM (Yield to Maturity)** - доходность к погашению (метод Ньютона-Рафсона)
- **Current Yield** - текущая доходность
- **Modified Duration** - модифицированная дюрация (влияние на цену при изменении ставок)
- **Convexity** - выпуклость (нелинейность цены)
- **Fair Price** - справедливая стоимость при различных сценариях доходностей

**Сценарии оценки:**
- Бычий сценарий (снижение ставок до 5%)
- Базовый сценарий (ставка 8%)
- Медвежий сценарий (повышение ставок до 10%)

### 3. Акции - Анализ по показателям компании

Комплексный анализ акций на основе фундаментальных показателей:

**Ключевые метрики:**
- **P/E (Price-to-Earnings)** - коэффициент цена/прибыль (недооценена/справедлива/переоценена)
- **ROE (Return on Equity)** - рентабельность собственного капитала
- **Дивидендная доходность** - текущая доходность по дивидендам
- **Payout Ratio** - процент прибыли, выплачиваемый в виде дивидендов
- **Устойчивость дивидендов** - оценка надёжности выплат

**Специальные функции:**
- **DDM Model (Gordon Growth)** - справедливая стоимость по модели дисконтирования дивидендов
- **Анализ риска** - оценка уровня риска (низкий/средний/высокий) на основе P/E, ROE, дивидендной политики
- **Потенциал роста** - расчёт ожидаемой доходности при целевых мультипликаторах
- **Маркер "экспортёр"** - для стратегии валютного хеджа (защита от ослабления рубля)

### 4. Фьючерсы - Модель Блека-Шоулза с 3 сценариями

Профессиональный анализ фьючерсов для трёх ролей трейдера:

#### A. Хеджер (Hedger)
Использует фьючерсы для страховки от неблагоприятного движения цены.

**Расчёты:**
- Справедливая цена фьючерса: F = S × e^(r×T)
- Хедж рацион (количество контрактов для полной страховки)
- Стоимость хеджирования
- Защита на 1% движение цены
- Сценарии P&L при различных движениях спот-цены

#### B. Спекулянт (Speculator)
Активно использует рычаг для максимизации доходности.

**Расчёты:**
- Количество контрактов с учётом рычага
- Требуемая маржа (margin)
- Максимальный убыток при заданном начальном капитале
- Три сценария P&L:
  - Бычий (рост на 1 волатильность)
  - Базовый (без движения)
  - Медвежий (падение на 1 волатильность)
- Ratio Risk/Reward

#### C. Арбитражер (Arbitrageur)
Ищет разницу между спот и фьючерс ценами (basis) для безрисковой прибыли.

**Расчёты:**
- Basis (разница спот-фьючерс)
- Fair Basis (справедливая разница)
- Arbitrage Mispricing (возможность арбитража)
- Стоимость операции (комиссии, спреды)
- Net Arbitrage (чистая прибыль минус комиссии)
- Рекомендация стратегии:
  - Cash-and-Carry (купить спот, продать фьючерс)
  - Reverse Cash-and-Carry (короткая продажа спота, покупка фьючерса)

### 5. Встроенные примеры портфелей

Три предкonfigured портфеля с реальными примерами инструментов Мосбиржи:

- 30% акции (Газпром, дивидендные истории)
- 10% кэш
- 5% спекуляции

- 35% облигации (ОФЗ, корпораты)

- 10% кэш
- 5% фьючерсы (хеджирование рубля)

**Агрессивный (25-40%+ годовых):**
- 15% облигации (подушка)
- 65% акции (Норильский никель, ростовые истории, Яндекс)
- 5% кэш
- 15% фьючерсы (спекулятивный блок)

## 🚀 Использование

### Создание портфеля
1. Нажать "Новый портфель"
2. Выбрать тип (консервативный/сбалансированный/агрессивный) или задать вручную
3. Портфель создан и готов к добавлению позиций

### Добавление позиции
1. Открыть портфель
2. Нажать "Добавить позицию"
3. Выбрать инструмент (облигация, акция или фьючерс)
4. Просмотреть детали и расчёты
5. Указать количество и цену покупки
6. Добавить в портфель

### Анализ инструмента
При выборе инструмента доступны вкладки:
- **Характеристики** - все расчёты и оценки
- **Добавить позицию** - форма для добавления в портфель

### Анализ портфеля
В портфеле три вкладки:
- **Позиции** - список всех позиций с P&L
- **Анализ** - распределение активов, анализ доходности
- **Характеристики** - информация о портфеле

## 📈 Математические основы

### Облигации - YTM (Yield to Maturity)
Используется метод Ньютона-Рафсона для поиска доходности:
- Решается уравнение: PV(купоны + номинал) = текущая цена
- Итеративный метод обеспечивает высокую точность

### Акции - DDM Model
Справедливая стоимость = D₁ / (r - g)
- D₁ - ожидаемый дивиденд на акцию в следующем году
- r - требуемая доходность
- g - темп роста дивидендов

### Фьючерсы - Блек-Шоулз
Справедливая цена = S × e^(r×T)
- S - спот-цена базового актива
- r - безрисковая ставка
- T - время до экспирации (лет)

## 🎓 Рекомендации по использованию

### Для консервативного инвестора
1. Начните с облигаций (ОФЗ, корпораты) как основу
2. Добавьте 20-30% дивидендных акций (Газпром, ЛУКОЙЛ)
3. Оставьте 10% кэша для возможностей
4. Мониторьте YTM облигаций - при снижении ставок цены растут

### Для активного трейдера
1. Используйте фьючерсы (Si, GZ, BRF) для спекуляций
2. Анализируйте сценарии Блека-Шоулза перед входом
3. Устанавливайте жёсткие стопы (выход при потере 2-3% капитала)
4. Не увеличивайте рычаг выше 2-3x

### Для дивидендной стратегии
1. Выбирайте акции с дивидендной доходностью 8%+ (ЛУКОЙЛ, ВТБ, МАГН)
2. Проверяйте Payout Ratio (оптимально 30-60%)
3. Анализируйте ROE (выше 15% - хороший знак)
4. Реинвестируйте дивиденды для компаундирования

### Для валютного хеджа (экспортёры)
1. Держите 30-40% портфеля в экспортёрах (энергетика, металлы)
2. Это естественная защита от ослабления рубля
3. При необходимости используйте фьючерсы Si как дополнительный хедж
4. Мониторьте basis Si для арбитража
